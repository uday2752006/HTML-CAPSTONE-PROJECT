# Roam Rides  
**Premium Motorcycles & Accessories E-Commerce Platform**  

Roam Rides is a responsive, luxury-themed e-commerce website for buying and selling premium motorcycles, scooters, and EVs.  
It features a **dark neon aesthetic** with a classic yet modern UI, designed for a high-end user experience.  

---

##  Features
- **Luxury Dark Theme** with neon highlights.
- **Responsive Design** (mobile, tablet, desktop).
- **Interactive Product Carousel** for featured bikes.
- **Modal-based Login System**.
- **Search Bar & Dropdown Menus** for easy navigation.
- **Product Listing Cards** with hover effects.
- **Bootstrap 5 Integration** for a modern grid system.
- **Font Awesome Icons** for a polished look.

---

##  Tech Stack
- **HTML5**
- **CSS3** (Custom Styling + Bootstrap 5)
- **JavaScript (Vanilla)**
- **Font Awesome** (Icons)
- **CDN-hosted Libraries** for faster load times
